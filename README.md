# AI-Native Study

## 목적

이 저장소는 AI 트렌드 소비가 아니라,
실제 Software Engineering / Backend 환경에서 AI-Native 방식을
연구·실험·구현하기 위한 스터디 저장소다.

스터디에서는 다음을 함께 다룬다.

- AI 기술 등장 배경과 구조적 변화 분석
- Agent / RAG / Tool Calling 구조 검증
- AI-assisted development workflow 실험
- 기존 서비스의 AI-Native 전환 방식 논의

## 핵심 방향성

단순 기능 소개보다 아래 질문을 중요하게 다룬다.

- AI가 Software Architecture를 어떻게 변화시키는가?
- 사람의 역할은 어떻게 변화하는가?
- 실제 현업에서는 어떤 방식으로 적용 가능한가?
